Content Delivery Network, även kallat CDN är ett system av servrar som är geografiskt utspridda och därmed kan snabbt leverera webbplatsinnehåll såsom HTML-sidor, bilder, JavaScript-filer m.m till användaren. Syftet med att de är geografiskt utspridda är för att det oavsett vart du befinner dig i världen ska gå snabbt att hämta webbplatsinnehållet eftersom att det inte behöver skickas över långa geografiska avstånd.

CDN fungerar genom att cachade lagras på flera servrar som är placerade på olika platser runt om i världen. När en användare besöker en webbplats som använder ett CDN dirigeras deras förfrågan till den server som är geografiskt närmast dem. Detta minskar avståndet som datan behöver färdas och minskar latensen och ger en trevligare användarupplevelse.

Fördelar med att använda ett CDN är just snabbare laddningstider men även ökad tillförlitlighet då det är mindre risk för avbrott eftersom innehållet är utspritt på flera servrar. Genom att lagra cachade kopior på CDN-servrar kan man också minska mängden datatrafik till den ursprungliga servern vilket kan sänka bandbreddskostnaderna. Många CDN-leverantörer erbjuder också säkerhetslösningar som kan göra det säkert att använda ett CDN.

Tjänster som kan använda sig av CDN är många. Streamingtjänster så som netflix och spotify använder sig utav CDN för att kunna distrubiera stora mängder data snabbt och effektivt. Även sociala medier så som instagram, facebook och tiktok använders sig utav CDN för att snabbt kunna ladda foton, videos m.m. Ett ytterligare användningsområde är olika chattplatformar så som whatsapp och discord. Där används CDN likt som de andra för att snabbt och utan latens leverera meddelanden, bilder och videos mellan användarna.
